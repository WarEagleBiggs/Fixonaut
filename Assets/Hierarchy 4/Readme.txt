(Middle Mouse) on name of GameObject for quick active/deactive.
(F2) with selected object(s) for multiple rename.
(Shift + W or S) for quick move GameObject order (sibling).
(Right Mouse) on tag or layer to edit tag or layer.
(Right Mouse) on component icon to display menu option.
(Middle Mouse) on component icon to open componnent properties.
(Hover Mouse) on scene name and press (F2) to rename scene.
(Right Mouse) on GameObject to display custom icon option.
Use "--->" prefix (Editable in setting) to mark GameObject as separator.
Instant Background feature work similar separator, need config in setting.
Open Project Settings -> Hierarchy to config this tool.
